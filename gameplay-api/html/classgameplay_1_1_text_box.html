Detailed Description

An editable text label. Tap or click within the text box to bring up the virtual keyboard.

Listeners can listen for a TEXT_CHANGED event, and then query the text box for the last keypress it received.

The following properties are available for text boxes:

label <labelID> { style = <styleID> alignment = <Control::Alignment constant> // Note: 'position' will be ignored. position = <x, y> autoWidth = <bool> autoHeight = <bool> size = <width, height> width = <width> // Can be used in place of 'size', e.g. with 'autoHeight = true' height = <height> // Can be used in place of 'size', e.g. with 'autoWidth = true' text = <string> }
