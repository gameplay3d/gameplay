Detailed Description

Defines a checkbox UI control. This is a button that toggles between two icons when clicked.

The following properties are available for checkboxes:

checkBox <checkBoxID> { style = <styleID> alignment = <Control::Alignment constant> // Note: 'position' will be ignored. position = <x, y> autoWidth = <bool> autoHeight = <bool> size = <width, height> width = <width> // Can be used in place of 'size', e.g. with 'autoHeight = true' height = <height> // Can be used in place of 'size', e.g. with 'autoWidth = true' text = <string> checked = <bool> iconSize = <width, height> // The size to draw the checkbox icon, if different from its size in the texture. }
