void gameplay::ScriptController::registerCallback ( const char *  callback,
const char *  function 
)

Registers the given script callback.

The 'callback' parameter must be one of the supported global callback event functions. The following strings are accepted: initialize, finalize, update, render, resizeEvent, keyEvent, touchEvent, mouseEvent, gamepadEvent. Signatures for the registered script function must match that of the corresponding signatures of these events on the Game class.

Parameters:
callbackThe script callback to register for.
functionThe name of the script function to register.
